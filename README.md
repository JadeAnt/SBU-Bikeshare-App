# SBU Bikeshare App
Sample implementation of the SBU Bikeshare App Project built with Flutter including Google Maps API and Geolocation integration.

## Description
Purpose of this is to create an application to allow for the access of bikestations on the SBU Campus without being physically present.

## Tasks
- Design and build a mobile application for the Bikeshare program at Stony Brook University
- Utilize the Google Maps API and Geolocation API in order to record the location of every bikestation on campus as well as the user's current location
- Create an algorithm for randomly generating a new passcode for the bikestation's electronic lock each time you wish to take out a bike
